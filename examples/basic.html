DiscoFlare Chat Demo

This is a basic example of the chat widget integration. Click the chat button in the bottom-right corner to start a conversation.

Note: This is a demo page. To use the chat widget on your website, you need to:
  1. Deploy the Cloudflare Workers (main worker and bot relay)
  2. Set up Discord bot and Turnstile
  3. Update the configuration below with your actual URLs and keys

Integration Code

<!-- Load Cloudflare Turnstile --> <script src="https://challenges.cloudflare.com/turnstile/v0/api.js" async defer></script> <!-- Load Chat Widget --> <script src="../widget/chat-widget.js"></script> <script> window.CloudflareChat = { workerUrl: 'https://your-worker.workers.dev', turnstileSiteKey: 'your-turnstile-site-key', theme: { primaryColor: '#5865F2', position: 'bottom-right' } }; </script>

Features

  • Real-time bidirectional communication via WebSocket
  • Session persistence across page refreshes
  • Automatic reconnection on connection loss
  • Spam protection with Cloudflare Turnstile
  • Thread reuse for returning visitors
  • Mobile-responsive design

Performance

  • Bundle size: ~5KB (minified)
  • Zero third-party dependencies
  • Perfect Lighthouse scores
  • No render-blocking resources
